# 2024-08-03 堀内から鈴木先生へ

## マインドフルネスの応用と草野様の幻視について

鈴木先生、始めまして。堀内寛己(ほりうちひろき)と申します。
こちらの角先生にお世話になっている引きこもりです。

草野様と入院時期と場所が重なり友達になり、禁を破って LINE を交換しました。

僕らは宗教観(仏教)が近いので、よく仏教に関する話をします。

僕がする仏教の話は、草野様のなぐさめになることが多いようです。

最近ではマントラ(仏教の呪文を表すカタカナ語)を取り入れた僕なりのマインドフルネス瞑想を、お互いに行っています。

ChatGPT に「統合失調症の症状が出ているとき瞑想は逆効果」と聞いていました。

しかし、せっかく覚えたマントラです。彼女が幻視に苦しんでいたときに、これが何かの役に立つかもしれないとの思いから

「瞑想ではなくただの深呼吸だ、と思ってマントラを 10 分唱えてごらん」と言いました。そうしたら彼女の幻視が止まりました。

# 2024-08-06 堀内から亜百合さんへ

## ぎゃーてー

般若心経に含まれる真実の呪文:
「ぎゃーてーぎゃーてー はらぎゃーてー はらそーぎゃーてー ぼじそわか」
のことを、ここでは縮めて「ぎゃーてー」と言います。

ちょっと難しくなりますが、般若心経の結びの現代語訳は:

----

それゆえに知るべきである。「智慧の完成」は大いなる真言であり、大いなる明らかな智慧の真言であり、
この上ない真言であり、すべての苦しみを鎮める。
それは、並ぶもののない真言であり、誤つことなきがゆえに真実である。
「智慧の完成」において、真言が唱えられる。
すなわち、

ぎゃーてーぎゃーてー はらぎゃーてー はらそーぎゃーてー ぼじそわか

(行った、行った、向こうの岸に行った。向こうの岸に完全に至った。悟りよ、幸いあれ!)

と。

----

となっています。

向こうの岸のことを、彼岸とも言います。彼岸花の彼岸ですね。

## ぎゃーてーと呼吸

- 息が続く場合は:  
  ぎゃーてーぎゃーてー(吸)、はらぎゃーてー(吐)のピッチで行ってください。
- 息が続かない場合は:  
  ぎゃーてー(吸)、ぎゃーてー(吐)、はらぎゃー(吸)、てー(吐)のピッチで行ってください。

## ぎゃーてー瞑想

- 心でぎゃーてーを唱えながら、それに合わせて呼吸してください。
- 雑念が湧いたら、それに気づいた自分を褒め、ぎゃーてーに戻ってください。
- 自分にとって一番楽な姿勢で行ってください。
- **幻視や、強い劣等感、強い罪悪感のある時はぎゃーてー瞑想はしないでください。**

## ぎゃーてー睡眠

- 就寝時に行うぎゃーてー瞑想です。時間を決めず、いつの間にか眠るまで瞑想します。
- **幻視や、強い劣等感、強い罪悪感のある時はぎゃーてー睡眠はしないでください。**

## ぎゃーてー呼吸

- これは瞑想ではない、ただの呼吸法だ、と自分に言い聞かせてから始めてください。
- 心でぎゃーてーを唱えながら、それに合わせて呼吸してください。
- 雑念が湧いたら、それを責めても良いし放置しても構いません。
- 自分にとって一番楽な姿勢で行ってください。

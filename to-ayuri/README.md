## 不快な症状

- どこかに幻視が出ている
- 足が震えている
- ラジオ体操の深呼吸で息切れがする
- 生まれてこなければよかったと、強く思う

いずれか一つでも当てはまれば、*不快な症状*があるとします。
当てはまるものが一つもなければ、*不快な症状*は無いとします。

## ぎゃーてー呼吸の方法

- これは瞑想ではない、ただの呼吸法だ、と自分に言い聞かせてから始めてください。
- 心でぎゃーてーを唱えながら、ただそれに合わせて呼吸してください。
- 自分にとって一番楽な姿勢で行ってください。
- 目は閉じるか、薄眼を開けてください。

## ぎゃーてーと呼吸のペース

- 息が続く場合は:  
  ぎゃーてー(吸)、ぎゃーてー(吐)<br>
  はらぎゃー(吸)、てーーー(吐)<br>
  はらそー(吸)、ぎゃーてー(吐)<br>
  ぼじそわ(吸)、かーーー(吐)<br>

  のペースで行ってください。

  なおこれは、ラジオ体操の深呼吸と同じペースです。

- 息が続かない場合は:  
  ぎゃー(吸)、てー(吐)、ぎゃー(吸)、てー(吐)<br>
  はらー(吸)、ぎゃー(吐)、てー(吸)、ーー(吐)<br>
  はら(吸)、そー(吐)、ぎゃー(吸)、てー(吐)<br>
  ぼじ(吸)、そわ(吐)、かー(吸)、ーー(吐)<br>

  のペースで行ってください。

  なおこれは、ラジオ体操の深呼吸の倍のペースです。

## ぎゃーてー瞑想の方法

- 心でぎゃーてーを唱えながら、それに合わせて呼吸してください。
- 雑念が湧いたら、それに気づいた自分を褒め、ぎゃーてーに戻ってください。
- 自分にとって一番楽な姿勢で行ってください。
- 目は閉じるか、薄眼を開けてください。
- ***不快な症状*がある時は、ぎゃーてー瞑想はしないでください。**

## ぎゃーてー睡眠の方法

- 就寝時に行うぎゃーてー呼吸・瞑想です。いつの間にか眠るまでぎゃーてー呼吸・瞑想します。
- ***不快な症状*がある時は、ぎゃーてー呼吸で行ってください。**
